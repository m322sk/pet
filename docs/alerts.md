# Система алертов

## Метрики для мониторинга
- MAU по месяцам (для набора данных — псевдо-дневные алерты считаются по месяцу).
- Оборот по всем категориям.
- Доля клиентов, активировавших кэшбэк.

## Логика
1. Рассчитываем метрики за текущий период.
2. Сравниваем с предыдущим периодом.
3. Если отклонение > 3σ — отправляем алерт.

## Реализация
- SQL-выражения в `sql/postgres_05_alerts.sql`.
- Отправка через cron/Airflow task + Telegram bot.

## Формат сообщения
```
[Cashback Alerts]
Период: 2023-07
MAU: 12 530 (-12.5% MoM) ⚠️
Оборот: 3.2 млн (-18.1% MoM) ⚠️
```
